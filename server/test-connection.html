🔌 MongoDB Connection Test

Use this page to test your MongoDB connection and API server.

📝 Notes:

  • Make sure the server is running: npm start in the server folder
  • Check your MongoDB credentials in server/.env
  • Your MongoDB URI has special characters - they need proper URL encoding

🔧 If Authentication Fails:

  1. Check MongoDB password is correct
  2. Verify URL encoding of special characters
  3. Confirm MongoDB server is accessible from your network
  4. Test with MongoDB Compass or another client
